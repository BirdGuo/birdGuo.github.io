File res/drawable/abc_vector_test.xml from drawable resource ID

May 9, 2017 | 问题记录

问题

今天碰到一个问题File res/drawable/abc_vector_test.xml from drawable resource ID，之前在测试的时候没测出来，今天在红米上突然出现了。

解决方案

  1. 调整support-v7包版本，具体见华为手机碰到的问题记录-NotFoundException: File res/drawable/abc_vector_test.xml from drawable resource ID
    但是这个解决方法我试了在我这并不管用，所以只能再换一种解决方法。

  2. build.gradle添加支持
    在app目录下的build.gradle android标签中添加

additionalParameters：添加附加参数
–no-version-vectors：添加低版本支持
参考问题Resources$NotFoundException: File res/drawable/abc_ic_ab_back_material.xml

深究

产生这个问题貌似是我用了添加了zxing库后在扫码页面使用到了Vector，然后自动生成了一个abc_vector_test.xml。
Vector是android L以后的一种矢量图支持
