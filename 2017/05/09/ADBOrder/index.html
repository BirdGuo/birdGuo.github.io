ADB命令备忘录

May 9, 2017 | 技术

直接调用ADB命令的

  • 安装app
    adb install path/XXX.apk
  • 卸载App
    adb uninstall package(app包名)
  • 获取root权限
    adb root
  • 获取读写权限
    adb remount

进入ADB终端调试
先进入cmd
一般都会先获取root权限和获取读写权限，也就是执行adb root和adb remount
然后输入adb shell，就会出现如下

  • 进入一个App
    am -n start package/activity路径
  • 显示所有已安装包
    pm list packages
  • 显示所有系统包
    pm list packages -s
  • 显示所有第三方包
    pm list packages -3

  • 查找所有有关的文件
    list -R|grep

  • 清除缓存
    进到data/app目录下，删除缓存文件
    cd /data/app
    rm -r package(包名)

    有些可能在system/app下，一般这里面都是系统自带程序，但不排除可能装到这里面了。
    cd system/app
    rm -r xxx.apk(打包后的apk名称，具体是神码可以看system/app中名称。保持一致即可)

  • 查看文件
    cat packages.xml
